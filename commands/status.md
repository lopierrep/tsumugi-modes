# /tsumugi-modes:status

Mostrale al usuario el estado actual de los modos corriendo el script `set-mode.py` con el tool Bash:

```bash
python "${CLAUDE_PLUGIN_ROOT}/scripts/set-mode.py" status
```

Mostrale la salida del script. Si querés, comentale brevemente qué implican los valores actuales (e.g., "auto_commit=false significa que voy a mostrarte mensajes propuestos pero no voy a commitear").
